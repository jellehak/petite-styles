# Sheet

The v-sheet component is a transformable piece of paper that provides a basic foundation. For example, properties such as rounded and shaped modify the border-radius property while elevation increase / decreases box-shadow.

# Cards

The v-card component is a versatile an enhanced version of v-sheet that provides a simple interface for headings, text, images, icons, and more
